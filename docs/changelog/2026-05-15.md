- 第一版价格型宏观闸门失败：收益被明显压低，回撤反而差于止盈候选。

## 下一步方案

目标仍然是长期年化 `15%`、Sharpe `1.0`，但验收必须分两层：

- 长样本全球代理：先把 `2006-2026` 推到年化 `11%-13%`、Sharpe `0.85-0.95`、最大回撤不劣于 `-18%`。
- 短样本 hybrid 实盘池：保留 `25%+` 年化潜力，同时最大回撤尽量压在 `-15%` 附近，且不能靠单资产满仓实现。

### 1. 做单资产上限稳定性扫描

候选参数：

- `max_position_weight`: `0.50 / 0.60 / 0.70 / 0.80`
- 同时跑：
  - `configs/global_etf_position_cap_yahoo.yaml`
  - `configs/bigquant_rotation_hybrid_position_cap.yaml`

验收标准：

- 全球代理不只单点有效，至少相邻两个上限参数都优于无上限策略。
- hybrid 短样本不能因为降仓过多导致年化跌破 `25%`。
- 优先看 walk-forward，而不是全样本。

失败标准：

- 只有 `0.60` 一个孤点好，其他上限都明显变差。
- 收益主要来自最近两年，早期 fold 没有改善。

### 2. 加资产桶预算，而不是只加单资产上限

当前风险：

- 单资产上限只能防止一个 ETF 满仓，不能防止同类资产聚集。
- 例如 `QQQ + SPY`、`恒生科技 + 恒生指数`、`黄金 + 原油 + 豆粕` 可能在相关性上升时一起回撤。

计划新增：

- `max_bucket_weight`
- `bucket_map`
- bucket 级别包括：
  - `CN_EQUITY`
  - `HK_EQUITY`
  - `US_EQUITY`
  - `COMMODITY`
  - `BOND_OR_CASH`

初始候选：

- 风险权益桶总上限：`0.80`
- 单一国家 / 区域权益桶上限：`0.60`
- 另类 / 商品桶上限：`0.60`
- 债券 / 现金不强制满仓，只作为风险预算剩余。

验收标准：

- 全球代理最大回撤进一步低于 `-16%`。
- hybrid 的 p95 最大权重和同桶暴露都下降。
- 年化换手不显著高于当前 position-cap 候选。

### 3. 把宏观风险闸门改成“仓位预算”，不是“风险开关”

本轮价格型闸门失败，原因是 risk-off 直接切防御资产太粗。

下一版不做二元开关，改成风险预算：

- risk score 高：风险资产预算 `100%`
- risk score 中：风险资产预算 `60%-80%`
- risk score 低：风险资产预算 `20%-50%`
- 剩余部分进入黄金、债券、现金或不配置。

可用信号分两类：

- 价格代理：
  - 全球风险资产均线宽度
  - 63 / 126 日宽基动量
  - 信用代理 `HYG / LQD`
- 真实滞后宏观：
  - HY OAS
  - 10Y-2Y 利差
  - 金融条件指数
  - 美元趋势

关键约束：

- 所有宏观数据必须滞后使用，默认至少 `21` 日滞后。
- 不允许同日宏观数据驱动同日仓位。
- 每天早上只使用前一交易日价格和已知宏观数据生成计划。

验收标准：

- 全球代理 walk-forward Sharpe 接近或超过 `0.85`。
- 2022 fold 和 2015-2016 fold 至少有一个明显改善。
- 如果降低回撤是靠长期空仓换来的，视为失败。

### 4. 做危机窗口归因

必须单独输出这些窗口的表现：

- 2008 金融危机
- 2011 欧债 / 美债降级
- 2015-2016 全球震荡
- 2018 加息与贸易摩擦
- 2020 疫情冲击
- 2022 加息杀估值

每个窗口记录：

- 区间收益
- 最大回撤
- 平均暴露
- 最大单资产权重
- 主要持仓资产
- 交易次数

目的：

- 找出策略到底怕什么。
- 避免只看全样本指标，把风险藏在少数年份里。

### 5. 机器学习只做轻量排序器和状态分类器

暂不做复杂模型。

第一版只允许：

- Logistic / Ridge / ElasticNet
- RandomForest 或 HistGradientBoosting 作为对照
- 输入因子必须来自前一日可获得数据。

两个任务：

- 横截面排序：预测未来 `20 / 60` 日相对收益排名。
- 风险状态分类：预测未来 `20 / 60` 日是否进入高回撤 / 高波动环境。

防过拟合规则：

- 必须 walk-forward 训练。
- 参数只能在 validation 段选。
- test 段只拼接样本外资金曲线。
- 若 ML 只在 `2020-2026` 有效、`2006-2026` 无效，则不进入主策略。

验收标准：

- 相比规则策略，全球代理 walk-forward 年化提升至少 `1.5` 个百分点，或最大回撤下降至少 `3` 个百分点。
- 交易频率不能明显上升。
- 特征重要性不能集中在单一资产的历史收益上。

### 6. 扩展实盘 ETF 池，但先用全球代理验证结构

短样本实盘池可以扩，但不能因为某个新资产回测好就加入。

候选方向：

- 国内权益：沪深300、中证500、创业板、科创类。
- 港股 / 中概：恒生指数、恒生科技、中概互联网。
- 海外权益：纳指、标普、日经、德国 / 欧洲，如有稳定 ETF。
- 商品：黄金、原油、豆粕，后续可看有色、煤炭等，但必须控制桶权重。
- 防御：货币 ETF、短债 ETF、国债 ETF，如数据和流动性稳定。

准入标准：

- 日线质量稳定，复权 / 折算异常可解释。
- 成交额足够个人实盘。
- 不新增强相关重复资产，除非用于替代低流动性标的。
- 必须映射到全球代理中的某个资产桶，先验证“桶”有效，再验证具体 ETF。

### 7. 下一轮执行顺序

建议按以下顺序推进：

1. 补一个 sweep 命令，批量扫描 `max_position_weight` 和未来 bucket 参数，输出统一表格。
2. 跑全球代理和 hybrid 双口径扫描。
3. 实现 bucket 权重上限，并复测。
4. 增加危机窗口报告。
5. 把宏观风险闸门改成连续风险预算。
6. 最后再试轻量 ML。

下一轮最值得优先做的是第 1 到第 4 步。它们能最快判断“风险结构优化”有没有真实增量，也能给后续 ML 和宏观因子一个更干净的基准。

- 全球代理 walk-forward 收益和 Sharpe 均弱于 position-cap 主候选；Hybrid 全样本年化跌破 `25%` 验收线，因此不能替代 `hybrid_position_cap` 主候选。
- 下一轮应把 bucket 预算改成参数扫描，而不是固定 `0.60 / 0.80`。

## 追加：数据口径修正后的新基线

发现问题：

- Yahoo 下载逻辑此前用 `adjclose` 写入 `close`，但 `open/high/low` 仍是未复权价格。
- 多数策略配置使用 `execution: next_open`，导致信号使用复权 close、成交收益使用未复权 open。
- 这会扭曲 global ETF 长样本结果，也会影响 hybrid 中来自 Yahoo 的部分标的。
- 多个 CSV 还包含 `Unnamed:*` 导出残留列。

修正：

- `MarketData.from_frame()` 读入时忽略 `Unnamed:*` 列。
- Yahoo 新下载数据按 `adjclose / close` 比例同步调整 `open/high/low/close`。
- 对历史缓存，如果检测到 OHLC 异常比例超过 `1%`，`next_open` 回测自动降级使用 close-to-close 收益，避免复权/未复权混用。
- `validate-data` 现在会输出行数、标的数、日期范围、重复行、`Unnamed` 列和 OHLC 异常比例。

## 下一轮优化方向：趋势状态预算

问题定位：

- Global 长样本 walk-forward 在 `2020` 前收益偏低，核心拖累不是全部早期样本，而是 `2013-2018`。
- `2012-02-06` 到 `2020-02-05` 样本外年化约 `3.94%`，Sharpe 约 `0.43`，最大回撤约 `-18.50%`。
- `2020-02-06` 到 `2026-05-11` 样本外年化约 `20.41%`，Sharpe 约 `1.27`。
- 低收益段里策略过于均衡 / 防守，平均持仓长期分散在 `GLD`、`TLT`、`IEF`、商品和 REIT 上，低配了当时最强趋势资产 `QQQ` / `SPY`。
- 快速消融显示：去掉止盈更差；`hold_num=3` 更低收益；单纯放宽 cluster 或加 score tilt 基本无效。因此问题不在单一 `max_position_weight` 或止盈参数。

下一步不要继续只调单资产上限，优先实现“趋势状态预算”：

- 当 `US_EQUITY` 趋势强且风险状态健康时，提高 `US_EQUITY` bucket 预算，允许 `QQQ` / `SPY` 更集中。
- 当风险状态转弱、波动上升或 2018/2022 类环境出现时，降低权益预算，把预算转给黄金、债券或现金。
- 初始信号可用：
  - `QQQ` / `SPY` 相对 `IEF` / `GLD` 的 63/126 日相对强度；
  - `SPY`、`QQQ` 是否在 120/200 日均线之上；
  - 组合趋势宽度；
  - `HYG/LQD` 或已有信用代理，必须滞后使用。
- 初始预算候选：
  - risk-on + US trend strong：`US_EQUITY` 上限 `0.80`，总权益桶上限 `0.90`；
  - neutral：`US_EQUITY` 上限 `0.50-0.60`，总权益桶上限 `0.70-0.80`；
  - risk-off：总权益桶上限 `0.20-0.40`，剩余进入黄金、债券或现金。

验收标准：

- Global walk-forward Sharpe 接近或超过 `0.90`，并尽量把年化推到 `11%-13%`。
- 不能显著放大 `2022` 回撤，最大回撤目标仍先控制在 `-20%` 附近或更低。
- `2013-2018` 段的年化必须明显改善，不能只靠 `2020` 后贡献。
- Hybrid 短样本不能因为过度风险预算而跌破 `25%` 年化。
